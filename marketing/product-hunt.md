## Description

StockFlow is a free, modern inventory management tool built for small businesses that are tired of tracking stock on spreadsheets.

**Why I built this:**
Small shops in Cambodia (where I'm from) still use paper and Excel to manage inventory. I wanted to build something simple, free, and powerful enough to replace that.

**Key features:**
- Add and manage products with SKUs, barcodes, photos, and categories
- Real-time dashboard with stock charts, value tracking, and alerts
- Low stock alerts — get notified before you run out
- CSV import — migrate from spreadsheets in one click
- Team access — invite employees with different permission levels
- Reports — inventory value, stock movement, category breakdowns
- Works on any device — install as a mobile app (PWA)
- Dark mode

**Tech:** Built with Next.js 14, Supabase, TypeScript, and Tailwind CSS. Deployed on Vercel.

**Pricing:** Free. No credit card. No trial. Just sign up and start tracking.
